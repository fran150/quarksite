$$.routing.activateHasher Método

Inicializa el sistema de ruteo de Quark.js. Permite definir una función customizada que se ejecute cada vez que el hash en la url cambia.

activateHasher([customRouter])

Parámetros

Nombre Tipo Descripción
customRouter Función Se puede especificar una función con la firma:
function(oldHash, newHash)
que se ejecutará cada vez que el hash de la página cambie.

Devuelve

Nada

Observaciones

Al invocar a esta función, Quark.js comenzará a escuchar cambios en el hash de la URL del navegador, con el hash ingresado rastreará la página que corresponde visualizar y y cargará los outlet con los componentes y configuración correspondiente.

Si se especifica una función en el parametro customRouter la misma deberá invocar a $$.routing.parse pasando el nuevo hash para que el enrutador realice el cambio de pagina.
