$$.routing.pages Método

Permite definir las páginas del aplicativo y sus correspondientes variables.

pages(pagesConfig [, paramsConfig])

Parámetros

Nombre Tipo Descripción
pagesConfig Objeto Objeto donde cada propiedad es el nombre de una página del aplicativo y el valor es a su vez un objeto donde cada propiedad es el nombre de un outlet y el valor el nombre del tag del componente que debe mostrarse en dicho outlet.
paramsConfig Objeto Objeto donde cada propiedad es el nombre de una página del aplicativo y el valor puede ser:
  • Un Array con el nombre de todas las variables que posee la página.
  • Un objeto donde cada propiedad es el nombre de la variable de la página y su valor inicial.

Devuelve

Nada.
