ko.associativeObservable Objeto

Quark.js extiende Knockout.js y agrega associativeObservables, estos son similares a los observableArrays solo que almacenan los items en pares clave valor. Basicamente contienen un objeto Javascript y permiten agregar, eliminar o editar pares de clave / valor.

Los associativeObservables se definen utilizando la función associativeObservable de Knockout, pudiendo pasar el valor inicial. var asoc = ko.associativeObservable({ 'id': 5, 'name': 'John', age: 20 });

Se puede leer o escribir el valor de los associativeObservables como con cualquier otro observable. // Returns the object { 'id': 5, 'name': 'John', age: 20 } var value = asoc(); // Sets a new value var value = asoc({ 'id': 1, 'name': 'Paul', age: 75 });

Asi como los observableArray notifican los cambios en la cantidad de elementos, los associativeArray notifican cuando se asigna un nuevo objeto al observable o se edita un valor a través de los métodos disponibles para tal efecto. Atención con esto, solo se notificará la modificación de elementos del objeto si dicha modificación se hace a través de los métodos disponibles a tal efecto.
