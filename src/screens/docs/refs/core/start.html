$$.start Método

Inicia Quark.js realizando el binding de la página con el modelo especificado.

start([model])

Parámetros

Nombre Tipo Descripción
model Objeto Objeto con el que se debe realizar el binding de la página principal sobre la que luego corre la aplicación.

Devuelve

Nada.

Observaciones

Setea la variable $$.started en true indicando que se inició la aplicación. Si se vuelve a invocar una vez iniciada la aplicación no hace nada.
