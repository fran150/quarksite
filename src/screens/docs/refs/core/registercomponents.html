$$.registerComponents Método

Permite registrar un conjunto de componentes agrupados en namespaces.

registerComponents(config)

Parámetros

Nombre Tipo Descripción
config Objeto Objeto donde cada propiedad es el nombre del namespace o tag y el valor de la propiedad es un string con el path al módulo web del componente o un objeto con esta misma estructura que define los componentes o namespaces hijos.

Devuelve

Nada.

Ejemplo:

$$.registerComponents({ "pages": { "home": "pages/home", "library": { "finder": "pages/library/finder", "details": "pages/library/details" }, "docs": { "home": "pages/docs/home", "installing": "pages/docs/installing" } } }); En este ejemplo se registran los componentes:

pages-home
pages-library-finder
pages-library-details
pages-docs-home
pages-docs-installing
