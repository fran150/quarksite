$$.emptyTemplate Método

Devuelve un template vacío. Util para definir componentes que son invisibles en el navegador como por ejemplo un origen de datos.

emptyTemplate()

Parámetros

Ninguno.

Devuelve

Un template de componente Quark.js vacío.

Ejemplo:

// The component doesn't load a template define(['quark', 'knockout'], function($$, ko) { function ExampleComponent(params, $scope, $imports) { var self = this; // Do something } // Pass the emptyTemplate return $$.component(ExampleComponent, $$.emptyTemplate()); });
