namespace binding

Permite definir un alias para un namespace y dentro del contenido del binding hacer referencia al namespace a través de dicho alias.

namespace: <fullName>, alias: <shortName>

Parámetros

Nombre Tipo Descripción
fullName string Nombre completo del namespace.
shortName string Alias del namespace dentro del elemento.

Observaciones

Este binding se suele utilizar en su forma virtual para hacer mas legible la vista de componentes que utiliza mucho componentes con nombres largos. Para nombrar un componente en forma abreviada se debe llamar al tag con <alias del namespace>:><nombre del componente>.

Puede obtener mas información sobre esto aqui

Ejemplo:

Si se tienen componentes con los siguientes nombres:

docs-sidebar-main
docs-sidebar-ctrl-link
docs-sidebar-ctrl-button
docs-sidebar-ctrl-search
docs-navbar-main
docs-navbar-ctrl-link
docs-navbar-ctrl-button
docs-navbar-ctrl-search

Se pueden utilizar con namespaces de la siguiente manera: <quark-component> <!-- ko namespace: 'docs-navbar', alias: 'nav' --> <nav:main> <nav:ctrl-link> <nav:ctrl-button> <nav:ctrl-search> </nav:main> <!-- /ko --> <!-- ko namespace: 'docs-sidebar', alias: 'sdb' --> <sdb:main> <sdb:ctrl-link> <sdb:ctrl-button> <sdb:ctrl-search> </sdb:main> <!-- /ko --> </quark-component>
