$$.isValidDate Método

Devuelve verdadero si la variable especificada es un objeto Date de Javascript y contiene una fecha válida.

isValidDate(variable)

Parámetros

Nombre Tipo Descripción
variable Cualquiera Variable que se debe verificar si es un objeto Date de Javascript y contiene una fecha válida.

Devuelve

Verdadero si la variable especificada es un objeto Date de Javascript y contiene una fecha válida.
