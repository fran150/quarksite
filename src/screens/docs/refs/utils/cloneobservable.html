$$.cloneObservable Método

Genera una copia del objeto especificado generando un objeto observable (que es aquel en donde todas las propiedades son observables).

cloneObservable(source)

Parámetros

Nombre Tipo Descripción
source Objeto Objeto del que se debe generar la copia

Devuelve

Copia observable del objeto especificado.

Observaciones

A diferencia del método clone este sí genera una copia de los observables en el nuevo objeto. Se debe utilizar principalmente para clonar elementos si se desea transformarlos en observables.
