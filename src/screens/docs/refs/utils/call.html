$$.call Método

Permite invocar a un callback pasando un número arbitrario de parámetros. Si el callback es un observable asume que dentro hay una función que es la que debe invocar.

call(callback [, param1[, param2[, ... param N]]])

Parámetros

Nombre Tipo Descripción
callback Observable o Función Callback (u observable que lo contiene) al que hay que invocar
params Cualquiera Luego del parametro callback se pueden especificar los parámetros que se deben pasar al callback.

Devuelve

Resultado del callback.

Observaciones

Sirve para invocar un callback que se haya definido al crear un componente. La función verifica que el callback esté definido y sea una función y en tal caso la invoca, sino no hace nada.

Ejemplo:

Al hacer click en un boton invoca al callback solo si el usuario del componente definio uno, pasando el item en donde se hizo clic y la cantidad de clics realizados hasta el momento.

define(['quark', 'knockout', 'text!./example.html'], function($$, ko, template) { function ExampleComponent(params, $scope, $imports) { var self = this; var clickCount = 0; $$.parameters({ onClick: undefined }, params, this); $scope.btnClicked = function(item) { $$.call(self.onClick, item, ++clickCount); } } return $$.component(ExampleComponent, template); });
