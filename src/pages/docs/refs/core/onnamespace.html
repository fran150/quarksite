$$.onNamespace Método

Permite registrar los componentes agrupandolos por namespaces para facilitar la lectura de los mismos.

onNamespace(namespace)

Parámetros

Nombre Tipo Descripción
namespace string Nombre del namespace

Puede encadenarse con los metodos:

  • namespace (para definir un nuevo namespace) dentro del actual,
  • con register para registrar un componente dentro del namespace actual,
  • con endNamespace para volver al namespace anterior

namespace(namespace)

Parámetros

Nombre Tipo Descripción
namespace string Nombre del namespace a definir dentro del actual

register(componentName, path)

Parámetros

Nombre Tipo Descripción
componentName string Nombre del componente a registrar dentro del namespace actual
path string Path al modulo web que define el componente.

endNamespace()

Parámetros

Ninguno

Ejemplo

Por ejemplo: $$.onNamespace('pages') .register('home', 'pages/home') .namespace('library') .register('finder', 'pages/library/finder') .register('details', 'pages/library/details') .endNamespace() .namespace('docs') .register('home', 'pages/docs/home') .register('installing', 'pages/docs/installing'); En este ejemplo se registran los componentes:

pages-home
pages-library-finder
pages-library-details
pages-docs-home
pages-docs-installing
