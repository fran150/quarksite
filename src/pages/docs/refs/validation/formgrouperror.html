formGroupError binding

Aplica la clase has-error a un elemento con form-group de Bootstrap cuando se produce un error en el observable o los observables especificados. Esto hace que tanto el input, como el label del grupo de elementos se muestre en rojo.

formGroupError: <field>

Parámetros

Nombre Tipo Descripción
field Observable o Array Se puede especificar un observable o un array de estos. Si se produce un error en alguno de los observables especificados aplica el estilo has-error

Observaciones

Acompañado del binding fieldError permite mostrar claramente al usuario el error producido en un formulario de entrada.

Ejemplo:

<div class="form-group" data-bind="fieldError: income"> <label for="incomeField">Enter your income:</label> <input type="text" name="incomeField" data-bind="value: income" /> <div class="text-error" data-bind="fieldError: income"></div> </div>
