waitReady binding

Bloquea el binding del contenido del elemento hasta que la dependencia con el nombre especificado este "lista". Se debe especificar el nombre con el que se "importó" la dependencia. Quark.js esperará para realizar el binding hasta que dicha dependencia se marque como "lista".

waitReady: <dependency>

Parámetros

Nombre Tipo Descripción
dependency string Nombre con el que se importó la dependencia.

Observaciones:

Este binding se suscribe a la señal readied del tracker asociado al componente y espera la llegada de un evento indicando que la dependencia con el nombre especificado está lista. Una vez marcado como "listo" procede a realizar el binding del contenido del elemento y se desuscribe de la señal.

Ejemplo:

Este ejemplo espera a que el componente employee-display este listo antes de mostrar la botonera para grabar o cancelar.

<h1>Employee Data</h1> <hr /> <employee-display data-bind="import: 'data'"> </employee-display> <div data-bind="waitFor: 'data'"> <button type="submit" data-bind="click: save">Save</button> <button data-bind="click: cancel">Cancel</button> </div>
