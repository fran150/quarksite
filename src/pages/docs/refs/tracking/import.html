import binding

Permite "importar" el modelo de un componente que se esta utilizando al modelo del propio componente. Se debe utilizar en el tag del componente a importar proporcionando el nombre con el que luego se desea acceder al mismo.

import: <name>

Parámetros

Nombre Tipo Descripción
name string Nombre con el que se debe importar el modelo del componente.

Observaciones

Quark.js envía como tercer parámetro al modelo del componente un objeto denominado tracker que permite acceder a los modelos de todos los componentes importados.

Para mas información acerca de como funcionan los imports, remitase a este articulo.

Ejemplo:

<quark-component> <h1>Componente Padre</h1> <component-example data-bind="import: 'example'"> </component-example> </component-example> </quark-component> define(['quark', 'knockout', 'text!./other-example.html'], function($$, ko, template) { function OtherExample(params, $scope, $imports) { var self = this; $$.wait($imports.ready, function() { var child = $imports.get('example'); child.someMethod(); }); } return $$.component(OtherExample, template); });
