Tracker.loaded Propiedad

Señal que se dispara cada vez que se terminó de cargar una dependencia. Debe tenerse cuidado ya que el hecho de que una dependencia se haya cargado no quiere decir que esté lista.

Ejemplo

<quark-component> <h1>Componente Padre</h1> <component-example data-bind="import: 'example'"> </component-example> </component-example> </quark-component> define(['quark', 'knockout', 'text!./other-example.html'], function($$, ko, template) { function OtherExample(params, $scope, $imports) { var self = this; $imports.loaded.add(function(name, model) { alert('Se ha cargado al dependencia ' + name); if (name == 'example') { model.someMethod(); } }); } return $$.component(OtherExample, template); });
