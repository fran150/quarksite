Tracker.readied Propiedad

Señal que se dispara cada vez que una dependencia pasa al estado "listo". Debe tenerse cuidado ya que el hecho de que una dependencia se haya marcado como "lista" no quiere decir que el componente este listo, puede haber otras dependencias que todavia no terminaron de cargar.

Ejemplo

<quark-component> <h1>Componente Padre</h1> <component-example data-bind="import: 'example'"> </component-example> </component-example> </quark-component> define(['quark', 'knockout', 'text!./other-example.html'], function($$, ko, template) { function OtherExample(params, $scope, $imports) { var self = this; $imports.readied.add(function(name, model) { alert('La dependencia ' + name + ' está lista'); if (name == 'example') { model.someMethod(); } }); } return $$.component(OtherExample, template); });
