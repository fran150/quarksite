$$.isDefined Método

Devuelve verdadero si la variable especificada está definida. Util para diferenciar entre un valor falso, (como puede ser 0, false o un string vacío) de un valor que realmente no está definido.

isDefined(variable)

Parámetros

Nombre Tipo Descripción
variable Cualquiera Variable que se desea verificar si está definida.

Devuelve

Verdadero si la variable especificada esta definida, falso en caso contrario.

Ejemplo:

define(['quark', 'knockout', 'text!./example.html'], function($$, ko, template) { function ExampleComponent(params, $scope, $imports) { var self = this; var notDefined; var empty = ''; var unTrue = false; alert($$.isDefined(notDefined)); // Returns false alert($$.isDefined(empty)); // Returns true alert($$.isDefined(unTrue)); // Returns true } return $$.component(ExampleComponent, template); });
