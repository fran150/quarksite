$$.clone Método

Genera una copia del objeto especificado.

clone(source)

Parámetros

Nombre Tipo Descripción
source Objeto Objeto del que se debe generar la copia

Devuelve

Copia del objeto especificado.

Observaciones

El metodo genera una copia del objeto y todas sus propiedades, si la propiedad almacena la referencia a otro objeto se copia la referencia, por lo que las propiedades de ambos (el original y el clonado) van a estar apuntando al mismo objeto en memoria.

Debido a esto, si el objeto tiene propiedades observables, el objeto clonado apuntará a los mismos observables, por lo que este método se debe utilizar preferentemente para clonar objetos planos (que no contienen observables).
