$$.makeDate Método

Toma el valor especificado e intenta convertirlo en un objeto Date de Javascript. Si el valor especificado no se puede convertir a una fecha válida y el parametro useToday es true devuelve la fecha de hoy, si el parámetro es false devuelve undefined.

makeDate(value[, useToday])

Parámetros

Nombre Tipo Descripción
value Cualquiera Valor que se debe convertir a fecha
useToday Boolean Indica si, en caso de que no se pueda convertir el valor a una fecha valida se debe devolver la fecha de hoy o undefined.

Devuelve

Objeto Date o undefined
