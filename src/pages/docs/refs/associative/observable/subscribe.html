ko.associativeObservable.subscribe Método

De la misma forma que los observables permite suscribir funciones que se debe ejecutar cuando el valor del associativeArray cambia. Cabe destacar que solo se notificará la modificación de un elemento si se hace a través de los métodos provistos para tal efecto.

subscribe(callback)

Parámetros

Nombre Tipo Descripción
callback función Función que se invocará cada vez que el valor se modifique. La firma de esta función es function(newValue)

Devuelve

De la misma forma que los observables devuelve el handler de la suscripción, y también aqui para evitar leaks de memoria se debe invocar al método dispose sobre este handler cuando se este eliminando el objeto que contiene el associativeObservable.
