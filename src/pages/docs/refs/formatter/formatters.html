$$.formatters Propiedad

Permite definir una serie de formatters que transforman un valor tanto al leer como al escribir de un observable. Se puede aplicar un formatter a cualquier campo a través del binding format.

Se pueden definir tantos formatter como sean necesarios, cada uno debe ser una propiedad del objeto formatters y se puede definir como un objeto o una función. Si se define como función la misma se ejecutará pasando como parámetro el valor a transformar y debe devolver el valor transformado aplicandose este formato en la lectura del campo.

function(value)

Si se define un objeto se puede definir el formato a aplicar al leer o al escribir en el campo. Para esto el objeto puede contener las siguientes propiedades:

  • read: function(value): Debe tomar el valor especificado en el parámetro value y devolverlo aplicando previamente el formato de lectura deseado. Esta función se ejecutará al leer el valor del binding. Se puede utilizar para, por ejemplo, formatear el valor a mostrar en pantalla.
  • write: function(value): Debe tomar el valor especificado en el parámetro value y devolverlo aplicando previamente el formato de escritura deseado. Esta función se ejecutará antes de escribir el valor del binding. Se puede utilizar para, por ejemplo, formatear el valor capturado en un input antes de enviarlo a un servicio.

Ejemplo

$$.formatters.moneySymbol = { read: function(value) { switch(value) { case "EUR": return "€"; case "USD": return "$"; } }, write: function(value) { switch(value) { case "€": return "EUR"; case "$": return "USD"; } } }
