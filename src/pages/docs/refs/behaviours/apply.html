$$.behaviours.apply Método

Aplica el o los comportamientos a un objeto.

apply(behaviour, object [, config])

Parámetros

Nombre Tipo Descripción
behaviour String o Array String o Array de strings con los nombres de todos los comportamientos a aplicarle al objeto.
object Objeto Objeto al que se le deben aplicar los comportamientos.
config Objeto Objeto con la configuración de el o los behaviours especificados.

Devuelve

Nada.
