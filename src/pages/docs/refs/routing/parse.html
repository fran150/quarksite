$$.routing.parse Método

Parsea la ruta especificada y cambia a la pagina correspondiente seteando ademas el valor de las variables de dicha página. Puede ser invocado desde cualquier objeto pero su uso mas común es dentro de la función customizada que se puede pasar al método activateHasher para disparar el tratamiento por parte de Quark.js de la nueva ruta.

parse(hash)

Parámetros

Nombre Tipo Descripción
hash string Hash que se debe parsear.

Devuelve

Nada.

Observaciones

Para mas información ingrese a este articulo

Ejemplo

El siguiente ejemplo extrae del hash especificado por el usuario la primer parte y asume que es el lenguaje, luego parsea el resto del hash.

$$.routing.activateHasher(function(oldHash, newHash) { var index = newHash.indexOf('/'); var language = newHash.substring(0, index); var hash = newHash.substring(index + 1); setLanguage(language); $$.routing.parse(newHash); });
