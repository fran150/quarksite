$$.redirectHash Método

Redirige el navegador al hash correspondiente a la página de Quark.js con el nombre especificado.

redirectHash(page [, config])

Parámetros

Nombre Tipo Descripción
page string Nombre completo de la página a donde se quiere redirigir el navegador.
config Objeto Objeto donde cada propiedad corresponde con el nombre de una variable, y el valor de la propiedad corresponde con el valor que se debe dar a dicha variable al generar la redirección.

Devuelve

Nada.

Observaciones

El uso de este método permite referenciar las páginas por su nombre interno facilitando la administración de los hash que componen las rutas de acceso a cada pagina o estado de la aplicación. En las aplicaciones Quark.js, este método es recomendado utilizarlo para los enlaces entre paginas internas de la aplicación. Para los enlaces externos se recomienda utilizar el método redirect.
