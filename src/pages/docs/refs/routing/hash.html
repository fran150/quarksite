$$.routing.hash Método

Devuelve el hash correspondiente a la pagina. Se debe especificar el nombre de la página y opcionalmente un objeto donde cada propiedad es el nombre de una variable de la pagina con el valor que se le debe asignar a dicha variable.

hash(page [, options])

Parámetros

Nombre Tipo Descripción
page string Nombre de la página.
options Objeto Objeto con el valor que se debe dar a las variables de la pagina.

Devuelve

Hash correspondiente a la pagina y variables especificados.

Ejemplo:

// For example, with a page named as main/client/new and route defined as client/{id}/new // Calling $$.routing.hash('main/client/new', { id: 5 }); // Will return client/5/new
