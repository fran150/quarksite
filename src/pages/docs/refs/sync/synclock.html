SyncLock Objeto

Este objeto es generado por la función lock de Quark.js. Permite sincronizar la ejecución de una o varias funciones con la ocurrencia de un evento. El objeto tiene dos estados, abierto y cerrado. Cuando se encuentra abierto las funciones que se suscriben se ejecutan inmediatamente, cuando está cerrado las funciones quedarán suscriptas y se ejecutarán en cuanto se abra.

Métodos

Nombre Descripción
lock Cierra el lock bloqueando la ejecución inmediata de las funciones que se suscriben.
unlock Abre el lock permitiendo la ejecución inmediata de las funciones que se suscriben y ejecutando las que se habian suscripto y estaban esperando la apertura.
isLocked Devuelve false si el objeto está abierto, true en caso contrario.
force Fuerza la apertura del objeto (lo abre sin disparar la ejecución de las funciones previamente suscriptas).
