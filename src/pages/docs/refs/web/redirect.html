$$.redirect Método

Redirige el navegador a la URL especificada.

redirect(url)

Parámetros

Nombre Tipo Descripción
url string Url a donde se desea redirigir el navegador.

Devuelve

Nada.

Observaciones

En las aplicaciones Quark.js, este método es recomendado utilizarlo solo para los enlaces externos. Para los enlaces entre paginas internas de la aplicación se recomienda utilizar el método redirectHash del sistema de ruteo, ya que permite referenciar las páginas por su nombre interno facilitando la administración de los hash que componen las rutas de acceso a cada pagina o estado de la aplicación.
