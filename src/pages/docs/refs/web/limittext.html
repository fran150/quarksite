$$.limitText Método

Corta la cadena en la cantidad de caracteres especificados, si el texto original es mas largo muestra puntos suspensivos al final del texto. Esta función es util para mostrar el resumen de un texto indicando que el mismo es mas largo. Se debe proporcionar al usuario alguna forma de poder ver el resto.

limitText(value, limit)

Parámetros

Nombre Tipo Descripción
value string String al que se quiere limitar el largo.
limit int Limite de caracteres de la cadena. Si el largo de la original excede este limite se reemplazarán los ultimos 3 por puntos suspensivos.

Devuelve

String cortado en el maximo de caracteres especificados.
