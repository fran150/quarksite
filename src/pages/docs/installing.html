Instalando

La forma mas facil de descargar e instalar Quark.js es utilizando Bower. Bower es un gestor de paquetes web que administrar las dependencias de tu aplicacion web, si aun no lo tienes instalado te recomendamos que lo hagas pues muchos de los modulos desarrollados para Quark.js se pueden descargar e instalar facilmente con esta herramienta. Puedes aprender mas ingresando a http://bower.io

Para descargar Quark.js y sus dependencias en tu aplicación abre una consola en la carpeta donde esta alojada y ejecuta: bower install quark Opcionalmente puedes incluir la opcion --save para agregar a Quark.js como dependencia a tu bower.json

Ejecutando el comando anterior descargará e instalará quark y todas sus dependencias dentro de la carpeta bower_components en el directorio de tu aplicación.
