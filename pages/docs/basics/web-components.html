Componentes Web

Los componentes web permiten la creación de elementos que pueden utilizarse en una pagina web para crear la interfaz de usuario o proveer de una funcionalidad especifica. Cada componente se asocia a un tag HTML, permitiendo agregar nuevos elementos personalizados a los que ofrecen los navegadores por defecto.

Estos componentes se utilizan de forma similar a como se usan los componentes HTML estandar, permitiendo su reutilización tantas veces como sea necesario.

En Quark.js un componente es básicamente un modulo web que contiene el código que describe la funcionalidad de dicho componente y una vista con lo que se debe mostrar al usuario en el navegador. Ambas elementos se relacionan con el patrón MVVM

Para definir un componente, se debe crear un modulo web que devuelva el resultado de la ejecución de la funcion component.
