Componentes Web

Los componentes web permiten la creación de elementos que pueden utilizarse en una pagina web para crear la interfaz de usuario o brindar alguna alguna funcionalidad especifica. Cada componente se asocia a un tag HTML, permitiendo agregar nuevos elementos personalizados a los que ya ofrecen los navegadores por defecto.

Estos componentes se utilizan de forma similar a como se usan los componentes HTML estandar, permitiendo su reutilización tantas veces como sea necesario.

En Quark.js un componente web es básicamente un Modulo Web que contiene el código, llamado modelo, que describe la funcionalidad de dicho componente y una vista con lo que se debe mostrar al usuario en el navegador. El modelo del componente y su respectiva vista se enlazan entre sí utilizando el patrón MVVM

Para la definición y administración de componentes web, sus correspondientes modelos y vistas, y el binding bidireccional de la vista con el modelo de cada componente Quark.js utiliza Knockout.js

Puedes consultar la documentación de Knockout si quieres saber mas acerca de esto.
