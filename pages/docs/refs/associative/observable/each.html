associativeObservable.each Método

Por cada elemento en el associativeObservable ejecuta la función especificada pasando como parámetros la clave y el valor de cada uno.

each(callback)

Parámetros

Nombre Tipo Descripción
callback función Por cada elemento contenido se invocará a esta función pasando clave y valor. La firma de la funcion es:
function(key, value)

Observaciones

Permite recorrer un associativeArray. var example = ko.associativeObservable({ 'id': 5, 'name': 'Paul', 'age': 30 }); example.each(function(key, value) { alert('The value of ' + key + ' is ' + value); });

Devuelve

Nada.
