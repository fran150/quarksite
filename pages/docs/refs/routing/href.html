href binding

Setea el atributo href de un link a la ruta correspondiente a la página especificada. Permite generar links utilizando el nombre de la página y no del hash asociado lo que facilita reapuntar todos los links modificando solo el mapeo.

href: <page> [, params: <pageParameters>]

Parámetros

Nombre Tipo Descripción
page string Nombre completo de la página a donde debe apuntar el link. Quark.js buscará en la tabla de mapeo el hash asociado a la página y generará el link correspondiente.
pageParameters objeto Si la página admite parámetros se debe especificar un objeto donde cada propiedad tiene el nombre de un parámetro y su valor es el valor que se desea dar en el link para dicho parámetro.

Ejemplo:

<!-- If page main/person/editor is mapped to hash person/{id}/edit --> <a data-bind="href: 'main/person/editor', params: { id: 5 }">Edit Person</a> <!-- Generates the hash #person/5/edit -->
