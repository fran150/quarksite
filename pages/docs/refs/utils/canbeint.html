$$.canBeInt Método

Devuelve verdadero si la variable especificada puede ser transformada en un entero válido. A diferencia de isInt devuelve verdadero si la variable es un string pero puede ser transformado en un entero válido.

canBeInt(variable)

Parámetros

Nombre Tipo Descripción
variable Cualquiera Variable que se desea verificar si es o se puede transformar en un entero válido.

Devuelve

Verdadero si la variable especificada puede ser transformada en un entero válido

Observaciones

Si desea verificar que la variable sea tipo entero utilice el método isInt

Ejemplo:

define(['quark', 'knockout', 'text!./example.html'], function($$, ko, template) { function ExampleComponent(params, $scope, $imports) { var self = this; var num = 5; var sNum = '5'; var str = 'marron 5'; $$.isInt(num); // returns true $$.isInt(sNum); // returns false $$.isInt(str); // returns false $$.canBeInt(num); // returns true $$.canBeInt(sNum); // returns true $$.canBeInt(str); // returns false } return $$.component(ExampleComponent, template); });
