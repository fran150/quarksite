$$.formatString Método

Reemplaza el valor de variables en una cadena de texto. Se puede utilizar de dos maneras:

formatString(str, object)

Por cada propiedad del objeto busca en la cadena un texto en la forma {<nombre propiedad>} y lo reemplaza por el valor de dicha propiedad en el objeto.

Parámetros

Nombre Tipo Descripción
str string String a formatear
object Objeto Objeto donde cada propiedad corresponde con una variable en el string.

Devuelve

String formateado.

formatString(str [, param1[, param2, [, .. param N]]])

Reemplaza las variables {0}, {1}, {2}, {3} ... {N}, por el primer parámetro luego del string, el segundo, el tercero, etc.

Parámetros

Nombre Tipo Descripción
str string String a formatear
params String Luego del parametro string se deben especificar los valores a reemplazar en el string.

Devuelve

String formateado.
