hasNotContent binding

Realiza el binding del contenido del elemento solo si el usuario no especificó cierto tipo de contenido en el componente.

Tiene 3 formas de uso:

hasNotContent

Realiza el binding solo si el componente no tiene contenido.

Parámetros

Ninguno.

hasNotContent: <selector>

Realiza el binding del contenido solo si el componente no tiene algun contenido que corresponda con el selector especificado.

Parámetros

Nombre Tipo Descripción
selector string Selector de jQuery que indica el contenido que se debe buscar en el componente.

hasContent: <name>, virtual: true

Solo realizará el binding del contenido si no se especificó algun tag virtual con el nombre especificado.

Parámetros

Nombre Tipo Descripción
name string Nombre del elemento virtual que no debe existir en el componente.
