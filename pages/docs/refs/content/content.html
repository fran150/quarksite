content binding

Permite al desarrollador del componente especificar donde mostrar el contenido definido por quién utiliza el componente.

Tiene 3 formas de uso:

content

Inserta todo el contenido especificado por usuario del componente.

Parámetros

Ninguno.

Ejemplo:

<quark-component> <div class="alert alert-danger" data-bind="content"> </div> </quark-component> Este componente muestra todo lo que se envíe como contenido dentro de un alert de Bootstrap. Por ejemplo, si se registra con el nombre bootstrap-alert se puede utilizar asi: <bootstrap-alert> Alerta! Esto es una prueba </bootstrap-alert>

content: <selector>

Inserta la parte del contenido especificado por el usuario que corresponda al selector jQuery especificado. Se puede utilizar para dividir el contenido en varias zonas como puede ser el encabezado y cuerpo de un panel.

Parámetros

Nombre Tipo Descripción
selector string Selector de jQuery que indica que parte del contenido especificado por el usuario del componente se debe insertar por el binding content

Ejemplo

<quark-component> <div class="panel panel-primary"> <div class="panel-heading" data-bind="content: 'bt-heading'"> </div> <div class="panel-body" data-bind="content: 'bt-body'"> </div> </div> </quark-component> Este componente utiliza el binding content para dividir el contenido utilizando selectores JQuery. El primer content selecciona todos los nodos que se encuentran dentro del tag bt-heading y lo muestra dentro del titulo de un panel Bootstrap, el segundo hace exactamente lo mismo pero el contenido del tag bt-body lo muestra en el cuerpo del panel. Por ejemplo, si se registra con el nombre bootstrap-panel se puede utilizar asi: <bootstrap-panel> <bt-heading> Titulo de ejemplo </bt-heading> <bt-body> Lorem Ipsum? </bt-body> </bootstrap-panel>

content: <name>, virtual: true

Buscará dentro del contenido especificado y se mostrarán solo los nodos que esten contenidos dentro de un tag virtual con el nombre especificado.

Parámetros

Nombre Tipo Descripción
name string Nombre del elemento virtual que contiene los nodos a insertar.

Ejemplo

<quark-component> <div class="panel panel-primary"> <div class="panel-heading" data-bind="content: 'bt-heading', virtual: true"> </div> <div class="panel-body" data-bind="content: 'bt-body', virtual: true"> </div> </div> </quark-component> Este componente utiliza el binding content para dividir el contenido utilizando selectores JQuery. El primer content selecciona todos los nodos que se encuentran dentro del tag bt-heading y lo muestra dentro del titulo de un panel Bootstrap, el segundo hace exactamente lo mismo pero el contenido del tag bt-body lo muestra en el cuerpo del panel. Por ejemplo, si se registra con el nombre bootstrap-panel se puede utilizar asi: <bootstrap-panel> <!-- bt-heading --> Titulo de ejemplo <!-- /bt-heading --> <!-- bt-body --> Lorem Ipsum? <!-- /bt-body --> </bootstrap-panel>
