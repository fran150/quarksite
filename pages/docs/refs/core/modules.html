$$.modules Propiedad

Associative Observable que contiene datos de todos los modulos cargados en Quark.js

La clave de cada item es el nombre con el que se registró el módulo y el valor es un objeto con las siguientes propiedades:

  • name: Nombre del modulo
  • path: Path en donde se encuentra el modulo.
  • info: Información del modulo web que contiene el modulo de Quark.js proporcionada por Require.js
  • config: Objeto de configuración del módulo
  • main: Objeto principal del modulo
