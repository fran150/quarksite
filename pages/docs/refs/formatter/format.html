format binding

Permite formatear un valor antes de pasarlo a un binding especifico como puede ser text o value. Quark.js permite definir una serie de formatters que definen como debe transformarse un valor tanto al leerlo como escribirlo hacia y desde un observable.

format: <target>, formatter: <name> [, bindTo: <name >]

Parámetros

Nombre Tipo Descripción
target todos Cualquier valor válido para el binding y que el formatter a aplicar pueda entender. Puede ser un valor plano o un observable.
formatter string Nombre del formatter a aplicar. Quark.js buscará en formatters una propiedad con ese nombre.
bindTo string String con el nombre del binding al que se debe aplicar el valor formateado. Por defecto se aplica al binding text

Observaciones

Se pueden definir transformaciones de lectura o escritura que se pueden aplicar con este binding para transformar el valor y aplicarlo a un binding tanto al leer como escribir, esto se hace a través del objeto formatters.

Ejemplo:

<input type="text" data-bind="format: totalIncome, formatter: 'money', bindTo: 'value'" /> <div data-bind="format: totalIncome, formatter: 'money'"></div>
