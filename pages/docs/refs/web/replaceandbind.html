$$.replaceAndBind Método

Reemplaza el contenido de un elemento por el HTML especificado y luego realiza el binding del nuevo contenido contra el modelo especificado.

replaceAndBind(placeholderSelector, html, model)

Parámetros

Nombre Tipo Descripción
placeholderSelector String, Object Objeto cuyo contenido se quiere reemplazar. Se puede especificar cualquier selector de jQuery válido o bien directamente la referencia al nodo DOM. Debe utilizarse con un selector que atrape un único elemento.
html String HTML con el nuevo contenido que se debe setear al elemento.
model Objeto Objeto contra el que se debe realizar el binding del nuevo contenido.

Devuelve

Nada.

Observaciones

Esta función es util cuando, por ejemplo, se está recibiendo contenido HTML desde un servicio Web y se desea mostrarlo en pantalla y realizar los bindings contra un objeto asociado.
