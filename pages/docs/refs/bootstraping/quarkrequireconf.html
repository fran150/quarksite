QuarkRequireConf Función

Devuelve la configuración de Quark.js que se debe aplicar a require. Se debe utilizar con la función requireConfigure

QuarkRequireConf()([bowerModulesDir])

Parámetros

Nombre Tipo Descripción
bowerModulesDir string Se puede especificar el directorio raiz en donde se encuentran los modulos bower, si no se especifica nada se toma el estandar: bower_components

Devuelve

Objeto con las configuraciones de Require para Quark.js.

Ejemplo:

var require; require = requireConfigure(require, QuarkRequireConf()); require = requireConfigure(require, { paths: { 'bootstrap': 'bower_components/bootstrap/dist/js/bootstrap.min', }, shim: { bootstrap: ['jquery'] } });
