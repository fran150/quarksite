requireConfigure Función

Permite combinar configuraciones de Require. Se puede utilizar esta función para combinar la configuración de Quark.js con otros módulos que su aplicación requiera. Esta función se encuentra dentro del script require.configurator.js que viene en la carpeta principal de Quark.js

requireConfigure(config1 [, config2])

Parámetros

Nombre Tipo Descripción
config1 Objeto Require necesita que especifique la configuración en la variable global require, puede pasar esta variable como primer parametro a sucesivas llamadas a esta funcion para ir agregando distintas configuraciones.
config2 Objeto Objeto con la configuración de Require que se quiere aplicar.

Devuelve

Objeto con las configuraciones de Require combinadas.

Ejemplo:

var require; require = requireConfigure(require, QuarkRequireConf()); require = requireConfigure(require, { paths: { 'bootstrap': 'bower_components/bootstrap/dist/js/bootstrap.min', }, shim: { bootstrap: ['jquery'] } });
