ko.observable.validate Método

Valida este observable. Se pueden suscribir las validaciones al observables para que se ejecuten cada vez que el valor cambia.

validate([subscribe])

Parámetros

Nombre Tipo Descripción
subscribe Boolean Indica si se deben suscribir los validadores al observable permitiendo que se ejecuten las validaciones cada vez que el valor cambie.

Devuelve

Verdadero el observable tiene un valor válido, caso contrario falso.
